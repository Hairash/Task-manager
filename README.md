# Task-manager
For now you can write commands in console to operate with app.  
They will not be saved after exit.

## Commands
* ```add```  
Add task to the database  
Special symbols:  
^ date  
= duration  
\# parent_id  

  Examples:  
  ```
  add Learn English
  add Try SkyEng ^31.08 =2h #1
  add Think about papers at school ^01.09 =30m #4
  ```
* ```list```  
Show all tasks
* ```exit```  
Exit the app
